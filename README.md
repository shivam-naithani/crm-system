# CRM System

A CRM (Customer Relationship Management) system built with Laravel.

🚧 Work in progress — features and UI will be improved.

## Setup

```bash
git clone https://github.com/shivam-naithani/crm-system.git
cd crm-system

composer install
npm install

cp .env.example .env
php artisan key:generate

php artisan migrate

php artisan serve
npm run dev
```
